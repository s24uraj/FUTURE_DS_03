📊 Marketing Funnel & Conversion Performance Dashboard
📌 Project Overview

This project focuses on analyzing marketing campaign performance using a structured funnel approach. The objective was to identify drop-off points, measure conversion rates, and generate business insights to improve overall campaign effectiveness.

This project was completed as Task 3 – Data Science & Analytics Internship at Future Interns.

The analysis was performed using the Bank Marketing Dataset (UCI Machine Learning Repository) and visualized in Tableau.

🎯 Business Objective

To answer key marketing performance questions:

Where are customers dropping off in the funnel?

What is the overall drop-off rate?

Which contact channels perform best?

Which job segments convert more effectively?

How does conversion trend over time?

🗂 Dataset Used

Bank Marketing Dataset
Source: UCI Machine Learning Repository

The dataset contains campaign-related customer information including:

Contact type

Job category

Month of contact

Campaign details

Conversion outcome (Yes/No)

🏗 Funnel Structure

The marketing funnel was structured as:

Contacted → Converted

Key KPIs calculated:

Total Contacts

Converted Customers

Conversion Rate (%)

Drop-Off Rate (%)

📊 Dashboard Components

The Tableau dashboard includes:

🔹 Drop-Off KPI

Overall drop-off rate

Dynamic status indicator (High / Moderate / Healthy)

🔹 Funnel Overview

Visual comparison of Contacted vs Converted users

🔹 Contact Channel Performance

Conversion rate by contact type (cellular, telephone, etc.)

🔹 Job Segment Performance

Conversion rate across customer job categories

🔹 Monthly Conversion Trend

Conversion rate trend across months

🛠 Tools & Skills Applied

Tableau (Data Visualization)

KPI Design

Business Insight Generation

Dashboard Interaction & Actions

📈 Key Insights

Overall drop-off rate is high, indicating opportunity for campaign optimization.

Certain job segments convert significantly better than others.

Conversion performance varies by contact channel.

Monthly trends highlight seasonality in campaign success.

💼 Business Recommendations

Focus campaigns on high-converting job segments.

Optimize contact channels with lower conversion rates.

Improve follow-up strategy to reduce drop-off.

Target high-performing months strategically.
